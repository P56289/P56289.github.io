Introduction

This customer management system allows performing basic actions on registered users, such as creating, listing, modifying, and deleting them. It includes three main modules:

  1. Customer Management: Initial window to navigate between the user creation and listing options.
  2. User Creation: Window to add a new user to the system.
  3. User Listing: Window to view, edit, and delete existing users.

Main Window

The main window acts as an access point for system functionalities. It includes the following actions:

Available Options

  1. Create UserOpens the window to add a new user.
  2. List UsersOpens the window that shows all registered users in the system.

User Creation

Access

From the main window, click the Create User button to open this functionality.

Creation Interface

The window allows entering the required information to register a new user. It includes the following fields:

  • Name: Enter the user's name.
  • Surname: Enter the user's surnames.
  • Email: Valid email address for the user.
  • Username: Unique username.
  • Password: Password to access the system.
  • Role: Select the user's role (e.g., Administrator, Technician).
  • DNI: National Identity Document, consisting of 8 numbers and 1 letter.
  • Date of Birth: Select the date of birth from a calendar.
  • Phone: 9-digit phone number.

Instructions for Creating a User

  1. Fill in all the mandatory fields.
  2. Click the Create User button.
  3. If any field is incomplete or contains errors, a message indicating the issue will be displayed. Correct the errors and try again.
  4. If everything is correct, a confirmation message with the created user's data will be displayed.

Important Validations

  • DNI: Must have the format 8 numbers followed by a letter (e.g., 12345678A).
  • Phone: Must be a valid 9-digit number (e.g., 987654321).
  • All fields are mandatory.

"Back" Button

Returns to the main window to access other functionalities.

User Listing

Access

From the main window, click the List Users button.

Listing Interface

The window shows a table containing all registered users. The available columns are:

  • ID: Unique identifier for the user.
  • Name: User's first name.
  • Surname: User's surname.
  • Email: Valid email address for the user.
  • Username: Unique username.
  • Password: Password to access the system.
  • Role: Select the user's role (e.g., Administrator, Technician).
  • DNI: National Identity Document, consisting of 8 numbers and 1 letter.
  • Date of Birth: Select the date of birth from a calendar.
  • Phone: 9-digit phone number.

Available Options

  1. wdit user

    2. Each row includes an Edit button, which allows modifying the selected user's data.

  2. Delete user

    3. Each row includes a Delete button, which removes the selected user after confirmation.

"Back" Button

Returns to the main window to access other functionalities.

Usage Flow

Create a User

  1. From the main window, select Create User.
  2. Fill in all fields and validate the entered information.
  3. Click Create User to save the record.

List Users

  1. From the main window, select List Users.
  2. View the list of registered users.
  3. To edit or delete a user, use the corresponding buttons in the Actions column.
